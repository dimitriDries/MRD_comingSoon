# Content structure — blocks to fill

Use this as a checklist when writing copy. Order follows the site flow.

---

## Meta / global

- [ ] **Page title** — Browser tab / SEO (e.g. "MonsieurData — Digital Analytics")
- [ ] **Meta description** — Single sentence for search results (if you add meta tags)

---

## Header

- [ ] **Logo** — Asset (see IMAGES_CHECKLIST.md)
- [ ] **Nav: Home** — Label
- [ ] **Nav: About** — Label
- [ ] **Nav: Services** — Label
- [ ] **Nav: How I Work** — Label
- [ ] **Nav: Insights** — Label
- [ ] **Nav: Contact** — Label
- [ ] **CTA button** — e.g. "Start a Project"

---

## Hero

- [ ] **Headline (h1)** — Value proposition in one line
- [ ] **Subtext** — Short paragraph (what you do / who it’s for)
- [ ] **Primary CTA** — e.g. "Start a Project"
- [ ] **Secondary CTA** — e.g. "See How I Work"

---

## About (teaser on homepage)

- [ ] **Section headline (h2)** — e.g. "Analytics without guesswork."
- [ ] **Paragraph 1** — Who you are, specialism, tools
- [ ] **Paragraph 2** — Focus / values (e.g. data integrity, maintainability)
- [ ] **CTA button** — e.g. "Learn more about me"

---

## About (overlay)

- [ ] **Overlay title (h2)** — e.g. "About"
- [ ] **Who I Am (h3)** — Heading
- [ ] **Who I Am** — 1–2 short paragraphs (background, focus)
- [ ] **What I Do (h3)** — Heading
- [ ] **What I Do** — Bullet list (services / offerings)
- [ ] **How I Work (h3)** — Heading
- [ ] **How I Work** — Bullet list (principles, approach)
- [ ] **What This Means For You (h3)** — Heading
- [ ] **What This Means For You** — Bullet list (benefits for client)
- [ ] **Closing line** — One sentence before CTA
- [ ] **CTA button** — e.g. "Start a Project"

---

## Services (teaser on homepage)

For each of the 4 services:

- [ ] **Service 1: title** — e.g. "Tracking Implementation"
- [ ] **Service 1: short description** — One line
- [ ] **Service 2: title**
- [ ] **Service 2: short description**
- [ ] **Service 3: title**
- [ ] **Service 3: short description**
- [ ] **Service 4: title**
- [ ] **Service 4: short description**
- [ ] **Section CTA** — e.g. "Explore All Services"

---

## Services (overlay)

- [ ] **Overlay title (h2)** — e.g. "Services"

For each service block:

- [ ] **Service title (h3)**
- [ ] **Who This Is For (h4)** — Bullet list
- [ ] **What's Included (h4)** — Bullet list
- [ ] **Deliverables (h4)** — Bullet list
- [ ] **CTA button** — e.g. "Discuss This Service"

---

## How I Work

- [ ] **Section headline (h2)** — e.g. "How I Work"
- [ ] **Step 1: title** — e.g. "Audit & Discovery"
- [ ] **Step 1: short text** — One sentence
- [ ] **Step 2: title**
- [ ] **Step 2: short text**
- [ ] **Step 3: title**
- [ ] **Step 3: short text**
- [ ] **Step 4: title**
- [ ] **Step 4: short text**
- [ ] **CTA button** — e.g. "Discuss Your Setup"

---

## Client logos

- [ ] **Section headline (h2)** — e.g. "Trusted by growing teams…"
- [ ] **Client 1** — Name / logo (see IMAGES_CHECKLIST.md)
- [ ] **Client 2**
- [ ] **Client 3**
- [ ] **Client 4**
- [ ] **Client 5**
- [ ] **Client 6**

---

## Testimonials (carousel)

- [ ] **Section headline (h2)** — e.g. "What clients say"
- [ ] **Testimonial 1: quote**
- [ ] **Testimonial 1: author name**
- [ ] **Testimonial 1: role**
- [ ] **Testimonial 1: company**
- [ ] **Testimonial 2: quote, name, role, company**
- [ ] **Testimonial 3: quote, name, role, company**  
  _(Add more entries in SectionTestimonial.jsx if needed.)_

---

## Insights

- [ ] **Section headline (h2)** — e.g. "Insights"
- [ ] **Insight 1: title**
- [ ] **Insight 1: short description**
- [ ] **Insight 1: URL** — "Read article" link
- [ ] **Insight 2: title, description, URL**
- [ ] **Insight 3: title, description, URL**
- [ ] **View all URL** — e.g. blog or insights index
- [ ] **View All Insights** — Button/link label

---

## Final CTA

- [ ] **Headline (h2)** — e.g. "Ready to fix your analytics foundation?"
- [ ] **Supporting text** — One short paragraph
- [ ] **Primary CTA** — e.g. "Start a Project"
- [ ] **Secondary CTA** — e.g. "Send a Message"

---

## Footer

- [ ] **Your name**
- [ ] **Location line** — e.g. "Remote / City, Country"
- [ ] **Email** — Mailto link
- [ ] **Short tagline** — One line (e.g. "Helping businesses…")
- [ ] **Services list** — Same 4 service names as nav (links to Services overlay)
- [ ] **Navigation list** — Home, About, Services, Insights, Contact
- [ ] **Social: LinkedIn** — Label + URL
- [ ] **Social: GitHub** — Label + URL
- [ ] **Social: Twitter/X** — Label + URL
- [ ] **Copyright line** — e.g. "© 2026 Dimitri — All rights reserved"
- [ ] **Privacy link** — Text (e.g. "Privacy Policy") + URL when you have the page

---

## Contact overlay

- [ ] **Overlay title (h2)** — e.g. "Let's talk about your analytics setup."
- [ ] **Form label: Name**
- [ ] **Form placeholder: Name** — e.g. "Your name"
- [ ] **Form label: Company**
- [ ] **Form placeholder: Company**
- [ ] **Form label: Email**
- [ ] **Form placeholder: Email** — e.g. "you@example.com"
- [ ] **Form label: Website**
- [ ] **Form placeholder: Website** — e.g. "https://"
- [ ] **Form label: Message** — e.g. "What do you need help with?"
- [ ] **Form placeholder: Message** — e.g. "Describe your analytics needs…"
- [ ] **Form label: Budget** — e.g. "Budget range (optional)"
- [ ] **Budget options** — Select range, Small project, Medium, Large / ongoing
- [ ] **Form label: Timeline** — e.g. "Timeline (optional)"
- [ ] **Timeline options** — Select timeline, ASAP, 1–3 months, 3–6 months, 6+ months
- [ ] **Submit button** — e.g. "Send Message"
- [ ] **Success message** — After submit (e.g. "Thank you. I’ll get back to you within 1–2 days.")
- [ ] **Response notice** — e.g. "I typically respond within 1–2 business days."
